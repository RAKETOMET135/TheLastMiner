# Controls

**Horizontal movement - A, D / Left Arrow, Right Arrow**
**Jump - W / Space / Up Arrow**
**Drop down - S / Down Arrow**
**Lift movement up - W / Space / Up Arrow**
**Lift movement down - S / Down Arrow**
**Inventory/exit - E**
**Interaction (sell, shop) - F**
**Mine - LMB (click / drag)**
**Buy - LMB (click)**
**Rescue - R**
**Rescue confirm - Enter**
